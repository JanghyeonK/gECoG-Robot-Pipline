############### Notice
We do not provide the executable file.

############### How to Use
- Must keep VLC folder together with main.exe!
- Run main.exe

############### Description
- [Load] : Load video
- [Play/Pause] : Play/Pause video
- [Save] : Save current created labels -> Video folder
- [Load Labels] : Load previously saved labels
- [Stats] : Check total time and count of labels

############### Shortcuts
- [Space] : Play/Pause video after pressing [Play] button
- [Number 1] ~ [Number 7] : Action label buttons
- [Arrow ←] [Arrow →] : Skip video
- [+] [-] : Video playback speed - Check speed at the top of screen
